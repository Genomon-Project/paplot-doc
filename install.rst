=================
Dependency
=================

 - python >= 2.7

Required web browser for viewing result file.
(Tested in the following)

 - Firefox 48
 - Google Chrome 44
 - Internet Explorer 11

=================
Install
=================

.. code-block:: bash
git clone https://github.com/Genomon-Project/paplot.git
cd paplot

python setup.py build install --user
